### 项目结构

- common 公用方法
- dal 数据模型
- service 服务接口
- api 项目接口层
- router 路由层


### IDE支持


- linux/mac eclipse导入执行
``` ./gradlew eclipse
```

- windows eclipse导入执行
``` gradlew.bat eclipse
```

### Gradle使用 [Gradle介绍文章](http://tech.meituan.com/gradle-practice.html)

- Gradle帮助
``` ./gradlew --help
```

- 任务查看
``` ./gradlew tasks
```

- 执行任务, eg: hello
``` ./gradlew -q hello
```

- 项目构建
``` ./gradlew build
```
